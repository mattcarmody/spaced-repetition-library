# Anki Ecosystem

[Anki](https://apps.ankiweb.net/) is a popular, largely free to use, largely open source spaced repetition software available on multiple platforms.
It is extensible with plugins and offers the ability to import and export decks as well as card templates.
Anki is very customizable, offering the ability to customize note templates as well as cards.
It is applied in many domains.
All of these factors and mroe combine to create a robust ecosystem around the platform.
This file attempts to compile many of those resources into one place, for my own use and that of others.

### Free Decks
The Anki website provides a hub for [sharing and downloading free decks](https://ankiweb.net/shared/decks) that others have made.
Topics vary widely, community members can share just about anything.

### Free Addons
The Anki website also provides a hub for [sharing and downloading free add-ons](https://ankiweb.net/shared/addons) that extend the base Anki functionality on desktop.
Functionality varies widely, community members build all sorts of things on the extensible base Anki software.

### Paid Decks / Addon Systems
[AnkiHub](https://www.ankihub.net/) provides a freemium deck & addon product that provides in depth medical decks and continuously updates them with the latest info,
crowd sourced from the community and vetted by their staff.

Fluent Forever offers [pronunciation trainers](https://fluent-forever.com/product/fluent-forever-pronunciation-trainer/) in many langauges for a one time fee of around $12 each.
